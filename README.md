#wiki death tracker

Отправляет на заданную почту сообщение о новом пункте в списке умерших на странице Википедии и ссылку
на страницу русскоязычной Википедии, если такая страница есть.
Если страницы на русском языке нет, отправляет ту ссылку, которая указана в исходной странице.

Для работы скрипта нужно создать в рабочей директории скрипта файл .env и 
указать в нем настройки почты:

SMTP_SERVER = smtp.yandex.ru

SMTP_USERNAME = your_sender_email@yandex.ru

SMTP_PASSWORD = your password

RECEIVER_EMAIL = recipient_email@yandex.ru


Для запуска скрипта используйте команду:

nohup python3 wiki_tracker.py &

Для остановки скрипта нужно найти его ID процесса (PID) и использовать команду kill.

Например: найти процесс по имени скрипта
ps aux | grep wiki_tracker.py

user  12345  0.0 0.1 123456 7890 pts/0 S 12:00 0:01 python3 wiki_tracker.py
где 12345 - это PID.

остановить процесс
kill 12345

Возможные улучшения скрипта: асинхронное выполнение запросов
